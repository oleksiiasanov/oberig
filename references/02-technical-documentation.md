## A. Що пристрій ТАКЕ (підтверджено посібником)

Oberig — це **SDR-пристрій** (software-defined radio) для **прийому, перегляду та аналізу
аналогових відеосигналів**. Це не просто «детектор з лампочкою»: він поєднує живий перегляд
відео, автоматичний сканер каналів і **спектральний аналізатор «Водоспад»**.

Ключова відмінність від типових детекторів категорії: окрім факту «є сигнал», пристрій дає
**живе декодоване відео**, **аналіз спектра** (бачить сигнали, заслабкі для декодування) і
показники якості сигналу **RSSI/SNR**.

## B. Режими роботи (5 застосунків, через Швидке меню) — підтверджено

1. **Перегляд (Live)** — прийом відео наживо.
   - Налаштування частоти кнопками ВЛІВО/ВПРАВО **кроком 1 МГц**.
   - Плашка з поточною частотою; автосинхронізація при достатньому сигналі.
   - Фото / запис відео кнопкою ЖИВЛЕННЯ (червона крапка під час запису).
   - **AGC** — автоматичне регулювання підсилення для найчіткішого зображення.
2. **Сканер** — автоматичний пошук активних передач у широкому діапазоні.
   - Відмічає «піки» з дійсним відеосигналом; список знайдених каналів.
   - Показує **RSSI** (потужність) і **SNR** (якість сигналу).
   - Автозбереження скріншота при кожній новій чіткій передачі (опційно).
3. **Водоспад** — спектрограма / теплова карта радіоспектра.
   - Бачить сигнали, навіть **заслабкі для декодування відео**.
   - Зум/ширина охоплення кнопками ВГОРУ/ВНИЗ — **до 8× ширше** одночасно.
   - Яскраві кольори (жовтий/червоний) = сильніші сигнали.
4. **Браузер** — перегляд збережених фото та відео.
5. **Налаштування** — конфігурація пристрою.

## C. Керування — підтверджено
- Фізичні кнопки: ЖИВЛЕННЯ (багатофункційна: коротко — назад/скріншот/фото-відео; довге 2 сек —
  вимкнення), МЕНЮ (Швидке меню з будь-якого екрана), ВВІД (підтвердження / вибір каналу),
  ВГОРУ/ВНИЗ (навігація, зум у Водоспаді), ВЛІВО/ВПРАВО (частота, параметри).

## D. Можливості та налаштування — підтверджено
- **Дві власні (кастомні) антени в комплекті, з автоперемиканням** на пороговій частоті
  («Перемикач антени 2»). Антени розроблені Oberig спеціально під кращий прийом сигналу на
  **нестандартних частотах**. _(Підтверджено власником продукту 2026-06-04; рамкуємо як інженерне
  призначення, без бенчмарк-цифр.)_
- **Налаштовуваний діапазон сканування** (початкова/кінцева частота в МГц задаються користувачем).
- Запис **фото та відео** на SD-карту (форматування у FAT32 з пристрою).
- Екран: тайм-аут (мс, 0 = не вимикати), активна/неактивна яскравість (0–1000), системні звуки/гучність (0–10).
- **Мова інтерфейсу: English / Ukrainian.**
- **OTA-оновлення прошивки** через мережу (USB-хост: Android або Windows ПК).
- **USB-C функції:**
  - **USB-камера (UVC)** — пристрій як «SDR Camera» у Zoom / OBS / VLC.
  - **Дистанційне керування частотою** з ПК (слайдер «Pan»).
  - **USB-мережа (CDC-NCM)** — спільний інтернет з хоста (для OTA).
  - USB Serial — консоль налагодження.
- Скидання до заводських налаштувань.
- Живлення/зарядка через USB-C (є «екран зарядки» при підключеному живленні).
